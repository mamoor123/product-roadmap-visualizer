# 🗺️ RoadmapFlow — Product Roadmap Visualizer

A modern, interactive product roadmap planning tool built with vanilla HTML, CSS, and JavaScript.

![RoadmapFlow Board View](roadmapflow-board-view.png)
![RoadmapFlow Analytics](roadmapflow-analytics.png)
![RoadmapFlow Light Theme](roadmapflow-light-theme.png)

## ✨ Features

- **📋 Board View** — Kanban-style columns with drag & drop
- **📅 Timeline View** — Chronological timeline of roadmap items
- **📝 Backlog** — Manage items before adding to the roadmap
- **📊 Analytics** — Visual charts for priority, category, and team workload
- **🌙 Dark/Light Theme** — Toggle between themes
- **🔍 Search** — Find items instantly with Ctrl+K
- **📤 Export/Import** — JSON export for backup and sharing
- **📱 Responsive** — Works on desktop, tablet, and mobile
- **💾 Auto-Save** — All data persists in localStorage
- **⌨️ Keyboard Shortcuts** — N to add item, Ctrl+K to search, Esc to close

## 🚀 Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Start planning your roadmap!

No build tools, no dependencies, no server required.

## 🎯 How to Use

### Adding Items
- Click the **"+ Add Item"** button in the top bar
- Click the **"+"** button on any column header
- Press **N** key when not in an input field

### Moving Items
- Drag and drop cards between columns
- Items automatically save their new position

### Switching Views
- **Board** — Kanban columns (Quarters, Months, or Now/Next/Later)
- **Timeline** — Chronological view of all items
- **Backlog** — Manage items not yet on the roadmap
- **Analytics** — Visual breakdown of your roadmap data

### Theme Toggle
- Click the moon/sun icon in the sidebar bottom

### Search
- Click the search box or press **Ctrl+K**
- Searches titles, descriptions, tags, owners, and categories
